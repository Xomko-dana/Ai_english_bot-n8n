# 🤖 AI English Course Bot (n8n)

This project is an AI chatbot built with n8n automation.

## 🚀 Features

* Telegram bot integration
* AI assistant (multi-language: EN / CZ / UA)
* Helps users choose English courses
* Simple sales conversation flow

## 🧠 Tech Stack

* n8n
* AI Agent
* Telegram Bot API

## 📂 How to use

1. Import `workflow.json` into n8n
2. Add your API keys
3. Activate workflow

## 💡 Use case

Automates communication with students and helps convert visitors into clients.
